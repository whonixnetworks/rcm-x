`rcm-x` ~ A collection of bash scripts aimed to easily mount [rclone](https://rclone.org) remotes to a unix/linux os.

----

## rcmnt  

> `rclone mount`

- Mounts a `rclone` remote to local file system

----

## rcmv

> `rclone move`

> [rcmv docs](rcmv/README.md)

- Uses `rclone move` to upload local files to a remote.
